# Brain Tumor MRI Classification

Deep Learning based **Brain Tumor MRI Classification** using EfficientNetB0 with **90% accuracy**.
This project predicts brain tumor types from MRI images using a trained deep learning model and provides a **Streamlit web application for real-time prediction**.

## Model Download

Due to GitHub file size limits, the trained model is hosted on Google Drive.

Download the model from here:
[Download Model](https://drive.google.com/file/d/1F7AfBngiMXLosK0iXxZNU4LSBjLipg5Z/view?usp=drive_link)

## Features

* MRI Brain Tumor Classification
* 4 Classes: Glioma, Meningioma, Pituitary, No Tumor
* EfficientNetB0 Deep Learning Model
* Streamlit Web App for Prediction
* Image Upload for Real-time Detection

## Tech Stack

* Python
* TensorFlow / Keras
* Streamlit
* OpenCV
* NumPy

## Dataset

Brain MRI Images for Brain Tumor Detection Dataset.

## Accuracy

Model Accuracy: **90%**
